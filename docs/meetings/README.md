# Meeting Documentation

This folder contains meeting agendas, notes, and related documentation for the sensor board project.

## Contents

- `MEETING_AGENDA_2026-01-13.md` - Project kickoff meeting agenda
- `MEETING_QUICK_REFERENCE.md` - Quick reference card for meetings

## Meeting Notes

Meeting notes should be saved here with the format:
- `MEETING_NOTES_YYYY-MM-DD.md`

## Structure

- **Agendas**: Prepared before meetings
- **Notes**: Taken during meetings
- **Action Items**: Tracked and followed up
